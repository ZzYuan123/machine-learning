# 第二章 线性回归（Linear Regression）

## 用途

预测连续值

例如：

- 房价预测
- 工资预测
- 气温预测

## 模型

y = wx + b

## 损失函数

MSE（均方误差）

作用：

衡量预测值和真实值之间的误差。

越小越好。

## 梯度下降

目的：

不断更新参数 w 和 b。

公式不用背。

理解：

> 顺着损失下降最快的方向不断更新参数。

## 学习率（Learning Rate）

学习率越大：

容易震荡。

学习率越小：

收敛慢。

# 第三章 正则化

为什么需要？

模型容易过拟合。

所以：

需要限制模型复杂度。

## Ridge（L2）

作用：

限制参数不要太大。

特点：

不会变成0。

## Lasso（L1）

作用：

限制参数。

特点：

可以把参数压成0。

可以做特征选择。

# 第四章 逻辑回归（Logistic Regression）

用途：

分类。

例如：

垃圾邮件识别。

## 为什么不用线性回归？

因为：

分类结果不是连续值。

所以：

引入：

Sigmoid

作用：

把任何值压缩到：

0~1。

表示概率。

## 判断方式

默认：

> 概率 > 0.5

预测为：

正类。

# 第五章 分类指标

## 混淆矩阵

TP

真正例。

FP

假正例。

TN

真负例。

FN

假负例。

## Precision（查准率）

预测正确的正样本比例。

公式：

TP / (TP + FP)

## Recall（召回率）

真实正样本找回来多少。

公式：

TP / (TP + FN)

## F1 Score

Precision 与 Recall 的调和平均。

# 第六章 决策树（Decision Tree）

思想：

不断提出问题。

例如：

X <= 2.5？

满足：

左边。

否则：

右边。

## Gini

基尼不纯度。

越小：

分类越纯。

0：

完全纯。

0.5：

最混乱。

## 常见字段

samples

节点样本数量。

value

各类别数量。

class

最终预测类别。

# 第十章 XGBoost

核心思想：

Boosting。

每棵树：

学习上一棵树的错误。

最终：

不断修正预测。

## Learning Rate

控制：

每次修正多少。

越小：

越稳定。

## Gamma

控制：

是否继续分裂。

越大：

树越简单。

## Lambda

控制：

叶子输出不要太极端。

作用：

防止过拟合。
